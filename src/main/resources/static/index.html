Rosa Server

Spring Boot example

Cette application est un petit exemple de ce que l'on peut faire rapidement avec Spring Boot. Le but est de proposer une API REST pouvant être appelée via une application Javascript et proposant plusieurs concepts de base comme des services REST sécurisés, des échanges via web sockets...

Pour découvrir l'API consultez la documentation Swagger

API Swagger

 

Fonctionnellement cette mini application permet de gérer votre conférence pendant laquelle vous allez inviter tous les speakers qui vous font rêver et qui vont venir pour présenter une session. L'API permet de gérer des conférences, des speakers et des participants. Ces derniers peuvent (quand ils sont connectés à l'application) voter pour les conférences et des websockets permettent de mettre toutes ces informations en temps réel pour les personnes voulant suivre les votes en direct.
